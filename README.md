# Lua-launcher-for-3ds
Homebrew app that runs in Lövepotion

To install,
simply install Lövepotion first.
Create a folder in your /3ds/ folder named Lualauncher, then create a folder within that folder named apps.
drag and drop the main.lua file within the Lualauncher.zip file in your game folder (or lovepotion folder, i just prefer game) in your /3ds/ folder.
finally, its as simple as putting your .lua files within that /3ds/Lualauncher/apps folder and opening Lövepotion.3dsx in homebrew launcher and booting your .lua file using the text based interface.
